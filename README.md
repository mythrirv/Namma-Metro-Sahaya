# Namma Metro Sahaya

Namma Metro Sahaya is a simple Android application developed using Kotlin in Android Studio to help first-time metro users travel easily and confidently using the metro system.

The application is designed mainly for:
- First-time metro users
- Rural users
- Elderly users
- People unfamiliar with metro routes and navigation

The app provides simple route guidance, bilingual language support (English and Kannada), and supports both dark mode and light mode.

---

## Features

- Metro Route Guidance
- Source and Destination Selection
- English and Kannada Language Support
- Dark Mode Support
- Light Mode Support
- Beginner-Friendly Interface
- Simple and Easy Navigation

## Installation Steps

### Step 1: Open in Android Studio

- Open Android Studio
- Click Open Existing Project
- Select the project folder

### Step 2: Sync Gradle

Wait for Gradle sync to complete automatically.

### Step 3: Run the Application

Option 1:
- Start Android Emulator
- Click Run

Option 2:
- Enable Developer Options
- Enable USB Debugging
- Connect Android device using USB
- Click Run
